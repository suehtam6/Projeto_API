# 🌍 API de Estados e Cidades

Esta API fornece dados geográficos brasileiros de forma rápida e organizada. É um projeto focado em **Backend**.

---

## 🚀 Funcionalidades & Deploy

- [x] **Deploy no Render:** API hospedada e pronta para consumo.
- [x] Listagem completa de todos os Estados brasileiros.
- [x] Consulta de Cidades vinculadas a cada Estado.
- [x] Filtros por siglas e códigos de identificação.

> **🔗 Link da API [Render](https://dashboard.render.com/project/prj-d77buafafjfc73ddp5bg)

---

## 🧪 Testes e Consumo

Para garantir a confiabilidade das rotas, utilizei o **Postman** para realizar testes de integração e validar as respostas JSON. Se você deseja testar a API, pode importar o link do deploy diretamente no seu cliente HTTP preferido.
